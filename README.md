# Phishing URL Detection System

# Overview
- Python-based cybersecurity tool
- Detects phishing and suspicious URLs
- Uses rule-based analysis
- GUI built using Tkinter



# Features
- HTTPS validation check  
- Suspicious keyword detection (login, bank, verify, etc.)  
- URL structure analysis (length, symbols like @, hyphen)  
- Website reachability check using HTTP response  
- Risk scoring system (Safe / Suspicious / Dangerous)  
- GUI interface using Tkinter  



# Tech Stack
- Python  
- Tkinter  
- Requests  


# Setup & Run

# Navigate to project folder
cd phishing-url-detector

# Install required dependency
pip install requests

# Run the project
python phishing_gui.py
